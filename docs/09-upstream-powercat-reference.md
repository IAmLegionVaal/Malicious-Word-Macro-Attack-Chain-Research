# Upstream Powercat reference

The working demonstration used a PowerShell Netcat-style script named `powercat.ps1`.

For the original upstream implementation and its documentation, see:

- [besimorhino/powercat](https://github.com/besimorhino/powercat)

The upstream project is an operational networking and shell tool. It is linked here as an external technical reference and is not copied into this repository.

Use it only on systems you own or are explicitly authorized to test, inside an isolated lab with no production credentials, bridged networking, shared folders, or unrestricted internet access.

This repository's screenshots, observed values, detection content, and replication guide explain how the tool fitted into the demonstrated Word → VBA → PowerShell → TCP listener chain.
