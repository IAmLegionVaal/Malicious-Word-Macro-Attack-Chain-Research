# Contributing

Contributions should improve defensive value.

Accepted examples:

- detection tuning,
- additional SIEM translations,
- false-positive analysis,
- hardening guidance,
- sanitized incident-response workflows,
- original lab screenshots,
- documentation corrections.

Not accepted:

- weaponized macros,
- live payloads,
- evasion instructions,
- persistence or credential-theft modules,
- operational command-and-control configurations.

Use a disposable lab and sanitized telemetry when validating changes.
